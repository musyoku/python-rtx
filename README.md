# three.py

C++ & Python 3D library inspired by [three.js](https://github.com/mrdoob/three.js/)

- [ ] Ray tracing

# Requirements

- pybind11
- Python 3
- OpenGL 4.5
- GLFW 3
- Ubuntu
- C++14 (gcc-6)

# Installation

**GLFW**

```
sudo apt install libglfw3-dev
```

**pybind11**

```
pip3 install pybind11 --user
```
