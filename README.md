# Requirements

- pybind11
- Python 3
- OpenMP
- Intel CPU
- Ubuntu
- CUDA
- NVIDIA GPU
- C++14 (gcc-6)

# Installation

**pybind11**

```
pip3 install pybind11 --user
```

**OpenMP**

```
sudo apt install libomp-dev
```
