# Python RTX

C++ & Python Ray Tracing library inspired by [three.js](https://github.com/mrdoob/three.js/)

# Requirements

- pybind11
- Python 3
- OpenMP
- Ubuntu
- CUDA
- NVIDIA GPU
- C++14 (gcc-6)

# Installation

**pybind11**

```
pip3 install pybind11 --user
```

**OpenMP**

```
sudo apt install libomp-dev
```
